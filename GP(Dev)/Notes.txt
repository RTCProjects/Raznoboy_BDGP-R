#Комментарии к версиям внутреннего ПО для блока детектирования гамма-поиска

1.0 Базовая версия штатного ПО

1.1
-Ограничение максимальной скорости счёта для 38-х счётчиков. Максимум ограничения задается параметром верхнего предела rangeMax

1.2(Dev)
-Корректирова рассчёта МД с использованием алгоритма динамического рассчета коэфициента эффективных открытых счётчиков
-Внедрение отладочного CAN протокола для группы блоков темы "Богомаз"
-Исправлена ошибка в алгоритме сортировки массива
-Изменение драйвера управления питанием счётчиков(добавлен "короткий" режим)
